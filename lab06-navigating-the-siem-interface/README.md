# 🧪 Lab 06: Navigating the SIEM Interface

## 📌 Lab Summary

In this lab, the Kibana SIEM interface was explored to understand its layout, navigation, and core features. Key sections including Dashboards, Discover, Alerts, Stack Management, and User Settings were accessed to gain familiarity with the platform used for security monitoring and log analysis.

---

## 🎯 Objectives

- Understand the layout of the Kibana SIEM interface
- Navigate dashboards and security alerts
- Explore configuration and management sections
- Access user profile and customization settings
- Change the Kibana interface theme

---

## 🛠️ Tools Used

- AWS EC2 (Ubuntu)
- Elasticsearch
- Kibana
- Google Chrome
- MobaXterm

---

## ✅ Task 1: Log in to the SIEM GUI

Accessed the Kibana web interface using the AWS EC2 public IP address.

**URL**

```text
http://EC2_PUBLIC_IP:5601
```

Successfully logged in using the **elastic** user account credentials.

## 📚 Key Concepts

| Feature | Description |
