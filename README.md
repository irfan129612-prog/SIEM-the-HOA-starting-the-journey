# 🛡️ Security Information and Event Management (SIEM) The HOA starting the journey

This repository documents my hands-on **Security Information and Event Management (SIEM)** learning journey through a series of practical labs. The labs focus on deploying, configuring, and managing an enterprise SIEM environment using the **Elastic Stack (Elasticsearch, Logstash, Kibana)** while integrating multiple log sources, security monitoring tools, and detection techniques.

The portfolio covers the complete SIEM workflow, including log collection, parsing, normalization, dashboards, alerting, correlation rules, incident investigation, threat intelligence integration, reporting, and basic automation.

---

## 🎯 Learning Objectives

- Deploy and configure an open-source SIEM platform
- Collect logs from Windows, Linux, and network devices
- Configure log parsing and normalization
- Build dashboards and visualizations
- Create correlation rules and security alerts
- Investigate security incidents
- Integrate IDS and Threat Intelligence feeds
- Perform log forensics and trend analysis
- Configure Role-Based Access Control (RBAC)
- Generate executive security reports
- Understand SIEM health monitoring and maintenance
